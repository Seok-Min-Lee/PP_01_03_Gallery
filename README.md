**[프로그램 4종 요약]**
- 촬영: 웹캠을 이용해 촬영한 이미지를 서버로 전송하고 체험 진행을 위한 고유 비밀번호를 표출
- 편집: 고유 비밀번호를 입력하여 촬영한 사진을 받고 포스트 프로세싱 필터와 페이스 마스크를 적용한 이미지를 다시 서버로 전송
- 전시: 편집된 이미지를 서버로부터 전달받아 화면에 표출
- 서버: 고유 비밀번호 및 이미지 관리
